# Codhak Pro — Real_world_project

Simple static website mockup for Codhak Pro (HTML & CSS).

## Overview

- Pages: `index.html`, `service.html`, `about.html`.
- Styles: `CSS/stylesheet.css`.
- Purpose: small business web design demo with contact/quote form and newsletter UI.

## Preview locally

1. Open `index.html` in your browser (double-click the file) OR
2. Serve from a local HTTP server (recommended) — in project folder run:

```bash
python -m http.server 8000
# then open http://localhost:8000 in your browser
```
